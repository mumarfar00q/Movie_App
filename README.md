🎬 Movie App – React Native

A modern movie browsing app built with React Native that lets users explore trending films, view detailed movie information, and discover actor profiles.

🚀 Features

Browse Trending, Upcoming, and Top Rated movies
View movie details, ratings, release dates, and cast
Explore actor profiles and filmography
Global search across TMDB database
Optimized UI for Android & iOS

🛠 Tech Stack

React Native
NativeWind (Tailwind CSS)
React Navigation (Stack)
Axios + TMDB API
React Hooks
